# FabledBazar 🧙‍♂️

Веб-витрина для продажи магических артефактов и макгаффинов. Проект представляет собой каталог товаров с подробными описаниями, вдохновленный мифами Лавкрафта, фэнтези и городскими легендами.

## О проекте

**FabledBazar** — это статический веб-сайт (с возможностью дальнейшей интеграции с бэкендом), который демонстрирует коллекцию уникальных предметов. Данные о товарах хранятся в структурированном CSV-файле, а визуальная часть построена на HTML, CSS и JavaScript.

Проект создан как демонстрационный или для использования в качестве основы интернет-магазина необычных товаров.

## Основные возможности

- 📋 **Каталог товаров** — вывод карточек артефактов с изображениями, описаниями и ценами.
- 🏷️ **Детальная карточка товара** — страница с полным описанием, характеристиками и ссылкой на покупку.
- 🎨 **Адаптивный дизайн** — стилизованный интерфейс с элементами фэнтези.
- 📦 **Гибкое хранение данных** — все товары описаны в одном CSV-файле, что упрощает редактирование и добавление новых позиций.
